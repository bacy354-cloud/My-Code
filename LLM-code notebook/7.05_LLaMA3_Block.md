## 本节目标

把前面学过的零件组装成一个 LLaMA-3 Decoder Layer。

本节重点：

1. LLaMA Block 的两个大模块是 Attention 和 MLP。
2. LLaMA 使用 Pre-Norm：先 Norm，再进 Attention / MLP。
3. 每个子层后都有残差连接。
4. Attention 负责 token 间交流，MLP 负责 token 内部特征加工。
5. SwiGLU MLP 用 `gate_proj`、`up_proj`、`down_proj` 三个线性层。
6. 整个 Decoder Layer 输入输出 shape 不变，都是 `[B, S, hidden_size]`。

## 核心直觉

05 是“组装阶段”。

前面已经学过：

```text
RMSNorm
SwiGLU
RoPE
MHA/GQA
KV Cache
```

这一节要回答：

```text
这些零件怎么拼成一层真正的 LLaMA Decoder Layer？
```

整体骨架：

```text
输入 x
 |
 |-- RMSNorm
 |-- Attention(RoPE + GQA + KV Cache)
 |-- 残差相加: h = x + attn_out
 |
 |-- RMSNorm
 |-- MLP(SwiGLU)
 |-- 残差相加: out = h + mlp_out
 |
输出 out
```

口诀：

```text
Norm -> Attention -> Add
Norm -> MLP -> Add
```

公式：

```text
h = x + Attention(RMSNorm(x))
out = h + MLP(RMSNorm(h))
```

## Pre-Norm

传统 Post-Norm 直觉：

```text
x -> 子层 -> 残差相加 -> Norm
```

LLaMA 的 Pre-Norm：

```text
x -> Norm -> 子层 -> 残差相加
```

直观理解：

```text
先把输入整理稳定，再送进 Attention / MLP。
```

这样在深层网络里更容易训练，梯度也更稳定。

## Attention 和 MLP 的分工

Attention：

```text
负责 token 之间交流。
```

也就是：

```text
token_i 可以根据注意力权重看 token_j。
```

MLP：

```text
负责每个 token 内部的特征加工。
```

更精确地说：

```text
所有 token 共享同一个 MLP 参数；
但每个 token 位置独立过这个 MLP。
```

所以可以记成：

```text
Attention = 横向交流，跨 token
MLP = 纵向加工，改 hidden 特征
```

## Shape 主链路

假设输入：

```text
hidden_states: [B, S, hidden_size]
```

Attention Block：

```text
residual = hidden_states
attn_input = input_layernorm(hidden_states)
attn_out = self_attn(attn_input)
hidden_states = residual + attn_out
```

shape：

```text
residual:      [B, S, hidden_size]
attn_input:    [B, S, hidden_size]
attn_out:      [B, S, hidden_size]
hidden_states: [B, S, hidden_size]
```

MLP Block：

```text
residual = hidden_states
mlp_input = post_attention_layernorm(hidden_states)
mlp_out = mlp(mlp_input)
hidden_states = residual + mlp_out
```

shape：

```text
residual:      [B, S, hidden_size]
mlp_input:     [B, S, hidden_size]
mlp_out:       [B, S, hidden_size]
hidden_states: [B, S, hidden_size]
```

所以整个 decoder layer：

```text
[B, S, hidden_size] -> [B, S, hidden_size]
```

形状不变，但内容被更新了两次：

```text
第一次：Attention 混入上下文信息
第二次：MLP 做非线性特征加工
```

## SwiGLU MLP

LLaMA 的 MLP 不是普通的：

```text
Linear -> GELU -> Linear
```

而是 SwiGLU：

```text
gate = SiLU(gate_proj(x))
up = up_proj(x)
hidden = gate * up
out = down_proj(hidden)
```

公式：

```text
MLP(x) = down_proj(SiLU(gate_proj(x)) * up_proj(x))
```

三个线性层：

```text
gate_proj: hidden_size -> intermediate_size
up_proj:   hidden_size -> intermediate_size
down_proj: intermediate_size -> hidden_size
```

以 notebook 测试参数为例：

```text
hidden_size = 512
intermediate_size = 1376
```

shape：

```text
x:              [2, 16, 512]
gate_proj(x):   [2, 16, 1376]
up_proj(x):     [2, 16, 1376]
gate * up:      [2, 16, 1376]
down_proj(...): [2, 16, 512]
```

为什么 intermediate_size 约为 `8/3 * hidden_size`：

```text
传统 FFN 常用 4 * hidden_size；
SwiGLU 有 gate 和 up 两条上投影；
为了让参数量接近传统 FFN，中间维度通常调到约 8/3 * hidden_size。
```

## 最终代码

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class DummyRMSNorm(nn.Module):
    def __init__(self, dim):
        super().__init__()
        self.w = nn.Parameter(torch.ones(dim))

    def forward(self, x):
        return x * self.w

class DummyAttention(nn.Module):
    def __init__(self, dim):
        super().__init__()
        self.proj = nn.Linear(dim, dim)

    def forward(self, x):
        return self.proj(x)

class LlamaMLP(nn.Module):
    def __init__(self, hidden_size: int, intermediate_size: int):
        super().__init__()
        self.gate_proj = nn.Linear(hidden_size, intermediate_size, bias=False)
        self.up_proj = nn.Linear(hidden_size, intermediate_size, bias=False)
        self.down_proj = nn.Linear(intermediate_size, hidden_size, bias=False)

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        return self.down_proj(F.silu(self.gate_proj(x)) * self.up_proj(x))

class LlamaDecoderLayer(nn.Module):
    def __init__(self, hidden_size: int, intermediate_size: int):
        super().__init__()
        self.hidden_size = hidden_size

        self.input_layernorm = DummyRMSNorm(hidden_size)
        self.self_attn = DummyAttention(hidden_size)

        self.post_attention_layernorm = DummyRMSNorm(hidden_size)
        self.mlp = LlamaMLP(hidden_size, intermediate_size)

    def forward(self, hidden_states: torch.Tensor) -> torch.Tensor:
        residual = hidden_states
        attn_input = self.input_layernorm(hidden_states)
        attn_out = self.self_attn(attn_input)
        hidden_states = residual + attn_out

        residual = hidden_states
        mlp_input = self.post_attention_layernorm(hidden_states)
        mlp_out = self.mlp(mlp_input)
        hidden_states = residual + mlp_out

        return hidden_states
```

更接近参考答案的紧凑写法：

```python
def forward(self, hidden_states: torch.Tensor) -> torch.Tensor:
    residual = hidden_states
    hidden_states = self.input_layernorm(hidden_states)
    hidden_states = self.self_attn(hidden_states)
    hidden_states = residual + hidden_states

    residual = hidden_states
    hidden_states = self.post_attention_layernorm(hidden_states)
    hidden_states = self.mlp(hidden_states)
    hidden_states = residual + hidden_states

    return hidden_states
```

学习阶段建议用展开版，因为更方便 debug shape。

## 易错点

1. `bias=False` 不影响 shape，但会影响模型结构和参数量。
2. `nn.Linear` 默认 `bias=True`，所以 LLaMA 风格必须显式写 `bias=False`。
3. `F.silu` 需要提前 `import torch.nn.functional as F`。
4. `residual` 保存的是子层之前的 hidden states。
5. 残差相加要求两边 shape 一样。
6. Attention 和 MLP 都要输出 `[B, S, hidden_size]`，否则残差连接会失败。
7. `pass` 放在 `return` 后不会执行，但学习代码里应删掉，避免干扰阅读。
8. `hidden_states` 是常见源码命名；学习时可以理解成当前流动的 `x`。
